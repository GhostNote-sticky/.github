# GhostNote — Portable Sticky Notes on Files & Folders

## About GhostNote

GhostNote provides a complete sticky note environment for users who want to attach notes directly to files and folders. GhostNote combines floating notes, file association, color coding, and search. GhostNote enables users to add reminders, to‑dos, or comments that appear on top of any file or folder. GhostNote runs on desktop platforms. GhostNote is trusted by designers, developers, project managers, and productivity enthusiasts worldwide.

## Deep Dive — Sticky Notes on Files Notes

Sticky notes: attach note to any file or folder, note floats above icon
Color coding: assign colors to notes (red, yellow, green, blue, purple)
Reminders: set due dates, notifications when file is opened
Search: find files by note content, search across all notes
Export: export all notes as JSON, backup, restore

## Quick Reference for GhostNote

| Operation | GhostNote Action |
|-----------|------------------|
| Add note to selected file/folder | Cmd+N |
| Edit existing note | Cmd+E |
| Delete note | Cmd+Delete |
| Change note color | Cmd+Shift+C |
| Set reminder | Cmd+Shift+R |
| Search notes | Cmd+F |
| Open preferences | Cmd+, |
| Show help | Cmd+Shift+/ |
| Quit GhostNote | Cmd+Q |
